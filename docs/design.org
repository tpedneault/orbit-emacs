#+title: visual identity

* Overview

Orbit Emacs ships two bespoke themes — =orbit-dark= and =orbit-light= — plus a
powerline modeline, a global header line, and a rich home-screen dashboard.
Every visual element is pure styling; no core feature is altered.

* Themes

| Theme          | Character                        |
|----------------+----------------------------------|
| =orbit-dark=   | Deep navy with warm amber accent |
| =orbit-light=  | Warm cream / paper               |

Toggle at runtime: =SPC t T=

Set the default in your config:

#+begin_src emacs-lisp
;; 'orbit-dark (default) or 'orbit-light
(setq orbit-user-orbit-theme 'orbit-dark)
#+end_src

* Powerline Modeline

The modeline uses colored segments separated by chevron arrows (=▶=).

#+begin_example
 N ▶ edit/myproject  my-file.el *  Emacs-Lisp          Git:main  42:8  14:05
 ─────┘ └──────────────────────────────────────────┘   └──────────────────────┘
 evil    left segments (amber bg → bg-dim)              right segments (bg-dim)
#+end_example

** Nerd Fonts separators

With a Nerd Font installed, enable filled arrow glyphs:

#+begin_src emacs-lisp
(setq orbit-user-nerd-fonts t)
#+end_src

This uses =\xe0b0= / =\xe0b2= instead of =▶= / =◀=.

* Header Line

A global header line is active in file, Dired, Magit, and org-agenda buffers.

#+begin_example
  ◉ edit/myproject  ›  src/components/App.tsx              ⏱ ABC-123  1:23
#+end_example

- Left: =◉= + context name (amber) + =›= separator + relative file path
- Right: active Org clock heading and elapsed time (only when clocking)

The header line is suppressed in the home buffer, terminal buffers, and the
minibuffer.

* User Config Reference

Add to =~/.orbit-emacs.d/config.el= to customise:

#+begin_src emacs-lisp
;; Starting theme ('orbit-dark default)
(setq orbit-user-orbit-theme 'orbit-dark)

;; Nerd Fonts powerline separators
(setq orbit-user-nerd-fonts nil)

;; Font overrides (auto-detected if nil)
(setq orbit-user-font-family "JetBrains Mono")
(setq orbit-user-font-height 140)
(setq orbit-user-variable-pitch-font "Inter")
#+end_src
